Publicar Betfree en GitHub Pages (móvil)

  1. En GitHub: Settings → Pages → Build and deployment. Origen: Deploy from a branch. Rama: main, carpeta: /docs. Guardar.
  2. En tu PC, con el digest ya corrido y betfree.db con datos: python scripts/export_pages_predictions.py
  3. Commit y push: git add docs/betfree_predictions.json docs/index.html y luego git commit / git push.
  4. Tras 1–2 minutos, abrí en el teléfono: https://<usuario>.github.io/<repositorio>/ (ej. MathiasAlejandr0.github.io/Betfree/).

Sobre appbetfree.html (raíz del repo): no es esta vista. Es un documento tipo “export de código” del proyecto ProBet (Next.js). El panel del modelo Betfree es index.html + betfree_predictions.json.

Automático al encender el PC (Windows)

  1. Configurá Git una vez: git config --global user.name "Tu Nombre" y git config --global user.email tu@mail.
  2. Hacé un git push manual exitoso (para que Windows guarde credenciales).
  3. En el repo, ejecutá: scripts\windows\instalar_betfree_al_encender.bat

Eso registra la tarea Betfree al encender PC: espera ~60 s, envía el digest por Telegram, exporta el JSON y hace git push si cambió la web. Logs en data/logs/betfree_startup.log. Probar sin subir a Git: powershell -ExecutionPolicy Bypass -File scripts\windows\betfree_al_inicio.ps1 -SkipGitPush
